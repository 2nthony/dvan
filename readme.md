## Introduction
Dvan now is a SPA generator powered by modern web tech, using Vue and webpack.

Check out the [documentation](https://dvan.evila.me).

## Installation
Recommanded to install dvan into your project:
```bash
cd your-project
yarn add dvan
```

## Usage
- `dvan dev` to run
- `dvan build` to build
